PyKQML
======

PyKQML is an implementation of KQML messaging in Python. It consists of the following classes:

    KQMLToken
    KQMLString
    KQMLList
    KQMLPerformative
    KQMLReader
    KQMLDispatcher
    KQMLModule

PyKQML can be installed as

    pip install pykqml

You can import KQML classes as, for instance,

    from kqml import KQMLList
